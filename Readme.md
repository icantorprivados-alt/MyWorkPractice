## 📌 Sobre este Repositorio

> **¡Bienvenidos, colegas docentes!** 👋

Este repositorio ha sido creado con el propósito de **centralizar y compartir** todos los recursos, ejercicios y prácticas desarrolladas durante el **Bootcamp de Tecnología organizado por MINEDUCYT**. Aquí encontrarán el material trabajado en cada sesión, organizado por módulo y tecnología.

🎯 **Objetivo principal:** Ser un espacio colaborativo donde los docentes-participantes del bootcamp puedan acceder, contribuir y aprender juntos, fomentando el intercambio de conocimiento entre colegas.

## 🤝 Cómo Colaborar

¿Quieres agregar tus prácticas o recursos al repositorio? ¡Es muy sencillo! Sigue estos pasos:

### Paso 1 — Clona el repositorio
```bash
git clone https://github.com/tu-usuario/bootcamp-mineducyt.git
cd bootcamp-mineducyt
```

### Paso 2 — Crea una rama con tu nombre
```bash
git checkout -b practica/nombre-apellido-tema
# Ejemplo: git checkout -b practica/juan-perez-flexbox
```

### Paso 3 — Agrega tus archivos
```bash
git add .
git commit -m "✨ feat: agrego práctica de Flexbox - Juan Pérez"
```

### Paso 4 — Sube tu rama y crea un Pull Request
```bash
git push origin practica/nombre-apellido-tema
```

> 💡 **Tip:** Recuerda organizar tu práctica dentro de la carpeta correspondiente al módulo.

## 📋 Normas de Uso

Para mantener el repositorio organizado y útil para todos, por favor sigue estas pautas:

- ✅ **Nombra tus carpetas** con el formato: `practica-XX-nombre-descriptivo`
- ✅ **Incluye un `README.md`** dentro de cada práctica explicando qué hace
- ✅ **Comenta tu código** para que tus compañeros puedan entenderlo fácilmente
- ✅ **Usa commits descriptivos** con emojis convencionales (`✨ feat:`, `🐛 fix:`, `📚 docs:`)
- ❌ **No subas archivos innecesarios** como `node_modules/`, `.env`, o archivos del sistema
- ❌ **No modifiques** las prácticas de otros compañeros sin consenso
